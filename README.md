## 🎯 界面特点

### 左侧导航栏
- **智能助手入口** - 绿色渐变按钮
- **患者选择器** - 下拉式患者切换
- **患者分组** - 术前患者、术后患者、历史患者
- **患者卡片** - 床号 + 姓名 + 状态标签
- **设置面板** - 固定在底部

### 右侧对话区
- **患者信息栏** - 姓名、诊断、住院号、床号
- **时间轴分隔** - 按日期分组对话
- **消息气泡** - 不同角色不同样式
  - AI 助手：蓝色背景卡片
  - 患者家属：白色气泡
  - 关键发现：带图标列表
- **建议卡片** - 带操作按钮（确认执行、修改方案、追问、开单建议）

### 底部快捷按钮
- 📄 生成病历
- 📝 开单建议
-  综合研判
- 👥 会诊
- 📊 查看趋势
-  交班摘要
- 💰 DRG 分析
